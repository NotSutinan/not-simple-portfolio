# Sutinan Not - Data Analyst Portfolio

A professional, modern-looking personal portfolio website for Sutinan Not, a Data Analyst. This project is a static website designed to showcase a professional profile, including a bio, selected projects, key skills, and contact information.

## 🚀 Features

- **Modern Dark Theme:** A sleek and professional design optimized for readability.
- **Theme Toggling:** Switch between dark and light modes according to your preference.
- **Responsive Design:** Fully responsive layout that looks great on mobile, tablet, and desktop screens.
- **JetBrains Mono Font:** Utilizing a clean, monospaced font for a distinct and professional feel.
- **Smooth Scrolling:** Easy navigation between different sections of the page.
- **Interactive Elements:** Subtle client-side enhancements for a better user experience.

## 🛠️ Built With

- **HTML5:** Semantic structure for better accessibility and SEO.
- **CSS3:** Custom styling using CSS Grid, Flexbox, and CSS Variables for theming.
- **JavaScript (Vanilla):** Client-side interactivity and theme management.
- **Google Fonts:** Featuring the 'JetBrains Mono' font.

## 📂 Project Structure

- `docs/`: Contains the live website files for GitHub Pages deployment.
  - `index.html`: The main entry point and structure of the website.
  - `style.css`: Contains all visual styling, including theme variables and responsiveness.
  - `script.js`: Handles theme switching, smooth scrolling, and other interactive features.
  - `images/`: Directory for project-related assets and screenshots.
- `plan/`: Documentation and project planning materials.
- `GEMINI.md`: Project overview and development conventions.

## 💻 Local Development

Since this is a static website, it doesn't require any compilation or complex setup.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/NotSutinan/not-simple-portfolio.git
   ```
2. **Open `docs/index.html`:**
   Simply open the `docs/index.html` file in your favorite web browser to view the site locally.

## 🌐 Deployment

This portfolio is designed to be easily deployed using **GitHub Pages**. 

1. Push your code to a GitHub repository.
2. Go to **Settings > Pages** in your repository.
3. Under the "Build and deployment" section, for **Source**, select **Deploy from a branch**.
4. Select your primary branch (e.g., `master`) and set the folder to `/docs`.
5. Save, and your site will be live at `https://<your-username>.github.io/<repo-name>/`.
